# 逃课模拟器 - 文案生成任务

请按以下要求，将三个 YAML 配置文件中的 `[TODO]` 占位符替换为正式文案。

## 配置文件位置

- `src/content/actionTexts.yaml` — 玩家行为描述文案
- `src/content/settlementTexts.yaml` — 每日结算描述文案
- `src/content/evaluationTexts.yaml` — 通关评价称号与评语

## 文案总则

1. **搞笑优先**：所有文案须轻松幽默，善用网络梗、大学生黑话、抽象文化表达
2. **绝不严肃**：不要出现说教、鸡汤、正式公告口吻
3. **篇幅紧凑**：单条控制在 40 字以内，保持节奏明快
4. **玩梗范围**：大学课堂、翘课、代课、通宵、外卖、舍友关系、教务处、考试突击等
5. **保留占位符**：`{cost}` 为金额占位符，不要删除或修改

## 1. actionTexts.yaml — 行为描述文案

该文件描述玩家各种行为的结果文字，会在白天推进和凌晨结算时展示给玩家。

### attend（老实上课）
| 字段 | 触发条件 | 要求 |
|------|---------|------|
| moodHigh | 心情 >= 70 | 上课效率极高的搞笑描述 |
| moodLow | 心情 <= 30 | 人在心不在的搞笑描述 |
| normal | 30 < 心情 < 70 | 正常上课的平淡描述 |
| sleepSuffix | 精力 <= 25 且随机触发 | 以空格开头，描述课上睡着 |
| phoneSuffix | 娱乐 <= 25 且随机触发 | 以空格开头，描述课上玩手机 |
| rollCallSuffix | 老师爱点名且被点到 | 以空格开头，描述被点到答"到" |

### skip（旷课）
| 字段 | 触发条件 | 要求 |
|------|---------|------|
| rollCall | 旷课且被点名 | 被逮到的搞笑描述 |
| successPool | 旷课且未被点名 | 4 条不同场景的旷课成功描述，引擎随机选一条 |

### subForOther（帮人代课）
| 字段 | 触发条件 | 要求 |
|------|---------|------|
| main | 帮人代课 | 替人上课赚外快的描述 |
| rollCallSuffix | 老师爱点名且随机触发 | 以空格开头，描述点到替的人名字时的反应 |

### hireSub（找人代课）
三条文案都需要包含 `{cost}` 占位符：

| 字段 | 触发条件 | 要求 |
|------|---------|------|
| fail | 15% 概率代课者也翘了 | 被双重背叛的搞笑场景 |
| rollCall | 代课成功且老师点名 | 花钱消灾的满意描述 |
| safe | 代课成功且未点名 | 轻松混过一节课的描述 |

### dawn（凌晨行为）
| 字段 | 行为 | 要求 |
|------|------|------|
| sleepEarly | 早睡 | 早睡早起精神好的描述 |
| gaming | 熬夜打游戏 | 游戏快乐但身体抗议的描述 |
| cram | 通宵赶作业 | deadline 驱动生产力的描述 |
| goOut | 出去浪 | 快乐但伤钱包的夜生活描述 |
| normalRest | 正常休息 | 平平无奇的夜晚 |

## 2. settlementTexts.yaml — 每日结算文案

| 字段 | 触发条件 | 要求 |
|------|---------|------|
| moodGood | 精力饱腹娱乐均值 > 期望值+3 | 一天过得很好的总结 |
| moodBad | 均值 < 期望值-3 | 一天过得很差的总结 |
| moodNormal | 均值接近期望值 | 平平无奇的一天 |
| creditWarning | 学分低于预警值 | 拼接在结算描述末尾，可含 ⚠️，教务警告语气但保持搞笑 |

## 3. evaluationTexts.yaml — 评价称号体系

### titlePool
每个评级（S/A/B/C/D）下有若干候选称号，`condition` 字段决定何时匹配：

- 支持的变量：`credits`（学分）、`mood`（心情）、`money`（金钱）、`roommateFavor`（舍友好感度）、`skipRate`（旷课率 0~1）
- 支持的运算符：`>=` `>` `<=` `<` `==` `and` `or`
- `"default"` 表示无条件兜底，必须放在该评级最后

要求每个评级至少 3 个候选称号（D 级可放宽），覆盖不同玩家画像：
- 学霸型（学分高、心情可能低）
- 搞钱型（金钱多、旷课率高）
- 社交型（舍友好感度高）
- 全面型（各项均衡）

### 其他字段
- `fallback`：所有称号都匹配不上时的兜底
- `failureHighCredits`：心情归零但学分尚可
- `failureLowCredits`：心情归零且学分极低
